  2. Changing a user's login Password
    Only an Admin user can change the login password for any given user at anytime. For all other users (ie., users other than the Admin user), most all fields (except the two password fields) are disabled. However, no matter your userlevel, you always have the ability to change your own login password at anytime by performing the following five steps:
    1) Click the Security tab if it is not already the current tab.
    2) Click the row in the grid containing your own username if it is not currently selected. (Note that your own username is automatically pre-selected the first time the Security tab is activated).
    3) Enter the new password into the New Password field.
    4) Enter the same new password into the Confirm Password field.
    5) Click the Save button.

  5. Logging out from the PAC Configuration Tool
    Once you are logged into the PAC Configuration Tool, you can choose to logout at any time by simply clicking the Logout button located at the top right of the page.
    Note that you can simply close the PAC Configuration Tool web page in your browser, and you will be automatically logged out after a few minutes, but clicking the Logout button, guarantees the best and most reliable overall security.
